# spiriter

AI 生成的用于 2D 游戏精灵图预处理与重排的 Web 工具。

## 功能

- 上传单张图像，自动识别并去除背景
- 支持背景容差范围（不是单一颜色）
- 使用“仅去除与边界连通区域”的策略，尽量避免误删人物内部颜色
- 自动拆分成多个独立角色对象
- 支持设置单格像素宽高、网格行列数的辅助框
- 自动处理后会将每个对象等比缩放到单个辅助框内并自动排版
- 支持网页端拖拽、缩放和鼠标滚轮缩放对象
- 支持克隆选中对象和删除选中对象
- 支持在自动处理后对对象局部执行连通透明擦除
- 支持基于当前排版结果生成精灵图表播放预览，并可控制帧率
- 支持在工作区内按住 Ctrl 加鼠标滚轮缩放工作区视图
- 支持在工作区内拖动画布视口浏览大尺寸辅助框区域
- 仅导出辅助框区域内的透明背景精灵图，不导出辅助框线

## 技术栈

- Python 3.11+
- FastAPI
- Pillow + NumPy
- 前端：原生 HTML/CSS/JS + Konva
- 依赖管理：uv

## 快速开始

```bash
uv venv
.venv\Scripts\activate
uv sync
uv run uvicorn app.main:app --reload
```

打开浏览器访问：

- http://127.0.0.1:8000

## 使用说明

1. 上传图片
2. 调整“背景容差”和“最小对象面积”
3. 点击“自动处理”
4. 设置辅助框的单格宽高、列数和行数
5. 自动处理后对象会按单格从左到右、从上到下自动排入辅助框
6. 如果对象数量超过预设行列，辅助框会自动向下续排并扩展导出区域
7. 你仍可继续拖拽、缩放每个对象，选中后可滚动鼠标滚轮缩放
8. 你可以克隆选中对象，或删除选中对象；删除也支持 Delete / Backspace
9. 可以开启“连通透明擦除”工具，然后点击对象中的某个位置，将该位置连通且颜色接近的像素擦成透明
10. 在工作区内按住 Ctrl 并滚动鼠标滚轮，可单独缩放工作区视图而不影响浏览器外部区域
11. 在工作区空白区域可直接拖动画布；也支持中键拖动画布视口
12. 需要检查动画时，可设置 FPS 并播放当前导出状态的精灵图表预览
13. 点击“导出精灵图”导出辅助框区域
14. 点击“下载导出结果”

## 算法说明（当前版本）

1. 从图像四周采样背景颜色
2. 按容差计算与背景色距离，得到候选背景
3. 仅删除和图像边界连通的候选区域，保护人物内部孔洞或近似颜色区域
4. 对剩余前景做连通域分割，得到独立角色对象
